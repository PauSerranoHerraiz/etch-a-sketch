# Etch-a-Sketch

Proyecto simple de dibujo en cuadrícula con colores aleatorios al pasar el ratón.

## Estructura
- [index.html](index.html)
- [styles.css](styles.css)
- [script.js](script.js)

## Uso
1. Abre [index.html](index.html) en tu navegador.
2. Usa **Change Size** para cambiar el tamaño de la cuadrícula.
3. Usa **Reset** para reiniciar la cuadrícula actual.
